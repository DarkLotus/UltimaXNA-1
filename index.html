UltimaXNA: Open Source Ultima Online Client in C#/XNA

UltimaXNA is an isometric massively multiplayer online role playing client that supports playing Ultima Online on your favorite freeshards! The client is already playable, but not all features are implemented. We are currently working on Milestone 0.7, which is a bugfix/minor feature push.

Watch a video of UltimaXNA Alpha at Milestone 0.4 (April 3, 2015)

Getting started

You should use Visual Studio 2013 to develop and compile UltimaXNA. You will also need to install XNA for Visual Studio 2013.

UltimaXNA does not include the Ultima Online resource files, and will not work on a computer that does not have Ultima Online installed. UltimaXNA is compatible with any Ultima Online client prior to the introduction of the UOP resource format, including Mondain's Legacy. Download the client files for Mondain's Legacy here.

UltimaXNA is designed to work with all server software. The recommended installation is RunUO 2.3, which you can download here.
